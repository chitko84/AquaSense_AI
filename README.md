## 📌 About

**AquaSense AI** is a web-based flood safety platform built for Malaysia. It combines a **Random Forest Machine Learning model** trained on Malaysian flood data from **2000–2010** with an interactive map, community flood reporting, shelter finder, and AI-generated safety guidance.

The system is designed to support users by providing flood risk insights, safer route suggestions, nearby shelter information, and simple emergency-safe recommendations.

Built with:

- **React**
- **Vite**
- **Firebase**
- **Leaflet**
- **Machine Learning Model**
- **AI Safety Guidance**
